ADMINS

  1. Clone This Repository
  2. Create a New GitHub Repository
  3. Push Cloned Repository to Your New GitHub Repository
  4. Add Your Partner as a Contributor

    5. Once The Contributor (Your Partner) Creates a Pull Request...

  5. Check New Pull Request
  6. Accept Pull Request
  7. Try Update Your Local Repository
  8. Fix Any Merge Issues If Necessary

CONTRIBUTORS

Once You Become a Contributor...

  1. Clone Admin's (Your Partner) New Repository
  2. Create a New Branch for this Repository
  3. Make Some Change to Your New Branch
  4. Push New Branch and Changes to Admin's Repository
  5. Create a Pull Request

    6. Once Your Pull Request is Accepted...

  6. Try Update Your Local Repository
  7. Fix Any Merge Issues If Necessary
